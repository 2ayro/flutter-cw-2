## تمرين 1

خطوات التمرين :
1. عين List  بالقيم التالية : [1, 4, 2, 7]
2. قيم بأنشاء متغير sum وقيمته الابتدائية 0
2.   قم بأستخدام  for loop بحيث تقوم بقرائة القيم الموجوده في List 
3. وعند كل قيمة قم بالتعديل على قيمة المتغير sum
4. قم بطباعة المجموع

